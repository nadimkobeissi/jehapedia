# Jehapedia

Jehapedia is a public encyclopedia of stories of Jeha, the legendary jester-like figure in Arabic folklore. Some of the stories are originals written by me, and marked as such in Jehapedia with a little green star on the navigation page, and with a green badge within the story page itself. All stories include English translations also written by me.

Jehapedia is available at [jehapedia.org](https://jehapedia.org).
